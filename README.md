# lock-in
An app that lets users set up focus sessions and personize their experience using custom themes and music.

## Features

- Set up a a pomodoro style timer for your focus session and breaks.
- Play calming music from a catalogue of focus music.
- Leaderboard ranks user based on their focus sessions.
- Rankings update @ 9pm everyday.
- Create a study group with your friends and lock in together!
- Get reports and analysis of your focus sessions.
